# Лабораторная работа №3 — Работа с базой данных SQLite

## Структура проекта

Task03/
├── make_db_init.py # Скрипт генерации SQL-файла и подготовки данных
├── db_init.sql # SQL-скрипт для создания таблиц и загрузки данных
├── db_init.bat # Батник, запускающий создание БД через sqlite3.exe
├── task3.bat # Основной батник: создаёт БД и выполняет запросы
├── users.txt # Датасет пользователей
├── movies.csv # Датасет фильмов
├── ratings.csv # Рейтинги пользователей
├── tags.csv # Теги фильмов

Для создания базы выполните: db_init.bat
После выполнения появится файл: movies_rating.db
Для выполнения запросов и вывода результатов используйте: task3.bat
